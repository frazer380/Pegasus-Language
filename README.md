# Pegasus-Language

To compile do:

```
python main.py main.p
```

Variables
```
var x = 5;
var array = ["Hello", "World"];
```

Printing
```
print "Hello World!";
print x;
print arr:array[0];
```

Input
```
var x = input();
print x;
```

If Else Statements
```
var x = 5;
var myArray = ["x is 5", "x is not 5"];

if x == 5: print arr:myArray[0];
else: print arr:myArray[1];
```
